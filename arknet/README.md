## arknet – Semantic Architecture Modeling for Domain-Driven Design

A semantic modeling platform for software architecture based on Domain-Driven Design.
Architecture models in RDF/OWL (W3C standards) instead of proprietary DSLs –
validatable (SHACL), queryable (SPARQL), and AI-ready (MCP).
