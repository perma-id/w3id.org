# KONI (Koiné Online Nexus of Integration) Namespace

- **Project URL:** https://github.com/kreeedit/KONI
- **Contact:** Tamás Kovács (https://github.com/kreeedit)
- **Description:** An Openly Licensed Linked Open Data Cross-Reference of the Thesaurus Linguae Graecae Canon.

This namespace provides stable, permanent identifiers for Ancient and Byzantine Greek authors and works, linking TLG identifiers to Wikidata, VIAF, and CTS URNs.
