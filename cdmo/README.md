# CDMO — Critical Decision-Making Ontology

Permanent identifier: **https://w3id.org/cdmo**

This directory configures the w3id.org permanent redirect for the CDMO ontology. Requests to
`https://w3id.org/cdmo` are content-negotiated:

- browsers / `text/html` → the HTML documentation,
- `text/turtle`, `application/rdf+xml`, `application/ld+json` → the ontology file (`cdmo.ttl`).

The ontology and its documentation are hosted at:
**https://github.com/dimitrisdoumanas19/cdmo** (served via GitHub Pages).
