# agrisemantics/

This is an umbrella namespace intended to provide persistent identifiers under:

- https://w3id.org/agrisemantics/

Resolution policy (HTTP redirects):
- https://w3id.org/agrisemantics/
  -> https://agroportal.lirmm.fr/ontologies/

- https://w3id.org/agrisemantics/{X}
  -> https://agroportal.lirmm.fr/ontologies/{X}

- https://w3id.org/agrisemantics/{X}/{PATH}
  -> https://agroportal.lirmm.fr/ontologies/{X}/{PATH}

Governance:
- Sub-namespaces `{X}` are expected to correspond to AgroPortal ontology acronyms or projects and are governed by AgroPortal policies, ensuring uniqueness and long-term stability.

Dereferencing:
- Resolution is performed via HTTP redirects. Content negotiation is handled by AgroPortal.
