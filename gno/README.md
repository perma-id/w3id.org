# Gno

Gno is an RDF vocabulary for managing RDF data in SPARQL triple stores. It extends [DCAT-R](https://w3id.org/dcatr) with classes and properties for services, stores, changesets, and commits.


GitHub: <https://github.com/rdf-elixir/gno>
