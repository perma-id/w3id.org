# /ogc

This [W3ID](https://w3id.org) provides a persistent URI namespace for the [HERITALISE project](https://heritalise-eccch.eu/).

It is maintained by the [Open Geospatial Consortium (OGC)](https://www.ogc.org) as part of the HERITALISE consortium.
