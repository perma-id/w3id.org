# w3id.org/nfdi4earth

This W3ID namespace provides persistent identifiers for resources and results of NFDI4Earth. 

# OneStop4All
[OneStop4All](https://onestop4all.nfdi4earth.de/) is the central web-based access point to all [NFDI4Earth](https://www.nfdi4earth.de/) resources and services. Its comprehensive search functionality enables the user to discover and access Earth System Science data, tools, services, training materials, and more from a wide variety of sources. Topics reach from research data management over data science approaches within and across sub-disciplines to assessing the FAIRness of data and repositories. The OneStop4All search leverages the [NFDI4Earth Knowledge Hub](https://knowledgehub.nfdi4earth.de/) as its semantic metadata backbone and also provides guidance on how to contribute as a data or software provider. 

# Persistent URI 
https://w3id.org/nfdi4earth redirects to https://onestop4all.nfdi4earth.de/. Signposting and RO-Crates are used to increase machine-actionability; therefore, some links resolve to HTML representations, while others resolve to JSON-LD metadata.
