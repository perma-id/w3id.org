# DPPROV

The DPPROV ontology serves to formally define the classes for the *Data Product Provenance*. The DPPROV ontology reflects key characteristics of the agents, activities and entites within a data product context.

Further information can be found on our [gitlab page](https://gitlab.com/tanja_a/dpprov/-/tree/main?ref_type=heads).
