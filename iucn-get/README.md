# /iucn-get/
This [W3ID](https://w3id.org) provides a persistent URI namespace for iucn-get resources.


## Uses
This namespace is used to provide URIs for the Metadata standards for the IUCN Global Ecosystem Typology (iucn-get) hosted in [Research Vocabularies Australia (RVA)](https://vocabs.ardc.edu.au).

The IUCN Global Ecosystem Typology is available at [global-ecosystems.org](https://global-ecosystems.org/) and the vocabulary will be documented in a GitHub repository and other sites (in preparation).

## Redirects
- https://w3id.org/iucn-get -> Research Vocabularies Australia (RVA)
