# GSORD w3id namespace

Persistent identifiers for the GSORD vocabulary.

Redirects all resources from  
`https://w3id.org/gsord/resources/...`  
to  
`https://imsmitesh.github.io/demo-vocab/gsord.germania-sacra.de/resources/...`.
