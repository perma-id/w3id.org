# AIEF — AI Ethics Assessment Framework Ontology

Permanent namespace for the AI Ethics Assessment Framework ontology developed as part of an MSc dissertation at Trinity College Dublin.

**Ontology IRI:** https://w3id.org/aief/

**Author:** Navina Ganapathy Amuthan, Trinity College Dublin (@navinaamuthan)

**Contact:** ganapatn@tcd.ie

**Description:** An ontology-driven knowledge graph integrating ethics assessment requirements across TCD REAMS, EU AI Act, Horizon Europe, and ACM/NeurIPS frameworks, mapped to EU Charter of Fundamental Rights articles.
