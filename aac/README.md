# /aac/ - Agentic Automation Canvas

This [W3ID](https://w3id.org) provides persistent identifiers for the **Agentic Automation Canvas (AAC)** project.

## Description

The Agentic Automation Canvas is a framework for designing and documenting agentic automation solutions. It provides:

- A structured approach to planning AI agent deployments
- JSON Schema for validating canvas documents
- RO-Crate profile for FAIR metadata packaging
