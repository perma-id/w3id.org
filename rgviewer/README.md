# rgviewer

This is a permanent identifier namespace for Research Groups Viewer.

Mantainer: 
- Name: Simone Angioni
- Email: simone.angioni@isti.cnr.it
- GitHub Username: simone-angioni

https://w3id.org/rgviewer/

rgviewer is a lightweight web application for browsing and visualizing Research Graph data integrating data from OpenAIRE Graph and ORCID. It provides a simple interface to explore entities, relationships, and metadata, making linked research resources easier to inspect, navigate, and share.
