# w3id.org/decision-kg

Permanent identifiers for the Decision-Knowledge Graph (DKG) vocabulary — a
linked-data vocabulary for representing energy-related decisions as first-class
knowledge-graph entities, with PROV-O provenance and SHACL conformance shapes.

Developed at the University of Ljubljana, Faculty of Computer and Information
Science. Source and versioned files: https://github.com/PouriyaMiri/decision-kg

| Identifier | Redirects to |
|---|---|
| https://w3id.org/decision-kg/v1 | vocabulary (303; content-negotiated: Turtle / JSON-LD context / HTML landing page) |
| https://w3id.org/decision-kg/v1/context.json | JSON-LD context |
| https://w3id.org/decision-kg/v1/shapes.ttl | SHACL shapes |
