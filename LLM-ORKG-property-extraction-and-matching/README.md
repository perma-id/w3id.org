# LLM-driven ORKG Properties Extraction, Matching and Evaluation Using GPT-3.5-Turbo

**Permanent Identifier:** https://w3id.org/LLM-ORKG-property-extraction-and-matching

**Repository:** https://anonymous.4open.science/r/LLM-ORKG-property-extraction-and-matching-FA51/

## Description
This repository contains the Python script developed as part of the study "LLM-driven Property Extraction for the Open Research Knowledge Graph: Advanced Prompt Engineering and Ontology Matching".

## Features
- Automated extraction of research dimensions using GPT-3.5-Turbo
- Evaluation against gold standard ORKG properties
- Semantic matching via ChromaDB + OpenAI embeddings
- SVG visualizations (heatmaps, bar charts)
