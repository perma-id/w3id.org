# Sappho Digital – Persistent Identifiers

This repository provides persistent HTTP identifiers for the [Sappho Digital](https://github.com/laurauntner/sappho-digital) project.

## Namespaces

The following persistent namespaces are defined:

- **Ontology**  
  https://w3id.org/sappho/ontology  
  → 303 redirect to RDF/Turtle

- **Vocabulary (SKOS)**  
  https://w3id.org/sappho/vocab  
  → 303 redirect to RDF/Turtle

- **Alignments**  
  https://w3id.org/sappho/alignments  
  → 303 redirect to RDF/Turtle

## Resolution

All namespaces resolve via HTTP 303 redirects to machine-readable RDF representations hosted at:

https://sappho-digital.com/documentation/

The project is RDF-first and does not currently provide HTML representations.
