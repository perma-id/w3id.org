# Exa-AToW Ontologies

HPC ontology: An ontology for HPC resources.
Workflow Ontology: Ontology for workflow interoperability.

Ontologies development is supported by [Exa-AToW](https://numpex.org/exaatow-architectures-and-tools-for-large-scale-workflows/), a targeted project by the [PEPR NumPEX](https://numpex.org).
