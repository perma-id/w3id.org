# SiMaT ontology

The Simulation of Material Testing Ontology, SiMaT ontology (simat) is a domain ontology in the field of material characterization through experimental testing and numerical simulation. It provides a common vocabulary for scientists and processes within the domain.
The ontology is applicable to documentation of scientific work and laboratory processes, standardization of terminology in Electronic Laboratory Notebooks (ELNs) and knowledge representation in modelling laboratory and computational processes.
