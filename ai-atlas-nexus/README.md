# AI Atlas Nexus

**Persistent identifier (PID) home** for IBM's AI Atlas Nexus ontology and related resources.

This directory hosts the redirection rules (via `.htaccess`) that resolve the PID
<https://w3id.org/ai-atlas-nexus> and its associated versions and vocabularies.

## 🔗 Resolvable Resources

The redirection rules point to the ontology documentation and files hosted on GitHub Pages:
**`https://ibm.github.io/ai-atlas-nexus/ontology/`**

### Main Ontology (Latest & Versioned)

The base PID <https://w3id.org/ai-atlas-nexus> redirects to the **latest** version.
