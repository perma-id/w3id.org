# /min

This [W3ID](https://w3id.org) namespace provides persistent IRIs for the
[MIN ontology repository](https://github.com/lepy/min-ontology).

## Canonical IRIs

- `https://w3id.org/min` -> current MIN ontology (`min.ttl`)
- `https://w3id.org/min/1.0.0` -> immutable release snapshot (`min-v1.0.0.ttl`)
