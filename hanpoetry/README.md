This namespace provides persistent identifiers for the Hanpoetry Linked Data dataset,
including ontology, SKOS vocabularies, and RDF datasets (Tang and Heian).
All RDF files are hosted in the GitHub repository:
https://github.com/TASYU78/Hanpoetry
