# /basicsemantics-owl/
This [W3ID](https://w3id.org) provides a persistent URI namespace for the normative (NEN) NTA 8035:2020 resources.

NTA 8035:2020 is Dutch national technical agreement (NTA) “_Semantic modelling of information in the built environment_”.

## Uses
This namespace contains 
[an RDFS and OWL ontology][terms].

[terms]: https://w3id.org/nta8035/def#

## Preferred Prefixes
```turtle
prefix nta8035: <https://w3id.org/nta8035/def#>
```
