# /ifcc/
This [W3ID](https://w3id.org) provides a persistent URI namespace for the IUPAC FAIR Chemistry Cookbook (IFCC)

## Uses
This namespace is used to mint GUPRIs for contributions to the IUPAC FAIR Chemistry Cookbook.
