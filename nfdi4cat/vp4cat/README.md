# vp4cat SHACL profile

**vp4cat** is a [SHACL](https://www.w3.org/TR/shacl/) profile for [voc4cat](https://github.com/nfdi4cat/voc4cat-tool) vocabularies used in the [NFDI4Cat](https://www.nfdi4cat.de/) project. It is closely based on the [vocpub profile](https://github.com/AGLDWG/vocpub-profile) developed by the Australian Government Linked Data Working Group (AGLDWG), with minor modifications to accommodate NFDI4Cat-specific requirements.
