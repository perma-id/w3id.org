Repository (italia/real-estate) for redirecting to ontologies and open data of Agenzia del Demanio (State Property Agency)  
===================

We intend to use **w3id.org/italia/real-estate** for national persistent URIs for state-owned real estate assets.

We initially plan to create:
+ `w3id.org/italia/real-estate/onto` for all the ontologies of the network mentioned above
+ `w3id.org/italia/real-estate/controlled-vocabulary` for all the controlled vocabularies
+ `w3id.org/italia/real-estate/data` for all the other types of data


Some redirect rules will be to the [github repository of Agenzia del Demanio](https://github.com/Agenzia-del-Demanio/NDC) we are using as semantic assets repo, open to all.
