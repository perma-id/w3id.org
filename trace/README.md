# /trace/

Persistent identifiers for the [TRACE project](https://transparency-certified.github.io/) (TRAnsparency CErtified).

TRACE develops standards for certifying that computational research results were produced as claimed. The primary vocabulary is **TROV** (Transparent Research Object Vocabulary), an OWL ontology for describing transparent research objects and the systems that create them.

## Redirects

| Path | Description |
|------|-------------|
| `/trace/trov/{version}` | TROV vocabulary (content-negotiated: HTML, Turtle, JSON-LD, RDF/XML, N-Triples) |
| `/trace/2023/05/trov` | Pre-release namespace (content-negotiated, same as versioned) |
| `/trace/trov/`, `/trace/`, etc. | Vocabulary index page (browsers); 406 Not Acceptable (others) |
