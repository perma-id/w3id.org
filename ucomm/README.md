# UCOMM Ontology

This w3id identifier provides a persistent URI for the UCOMM (Unified Cataloguing Ontology Model for Manuscripts) ontology.

## URI
https://w3id.org/ucomm

## Description
This identifier redirects to the RDF/XML version of the ontology and to its HTML documentation via LODE.

The ontology is hosted on GitHub : https://github.com/fazia97/ucomm-ontology
