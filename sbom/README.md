# w3id.org/sbom

Persistent URI namespace for the **SBOM Bridging Ontology for Mapping** -
a SKOS ontology and [SSSOM] crosswalk files that connect
Software Bill of Materials (SBOM) minimum-element standards to
exchange formats.

[SSSOM]: https://mapping-commons.github.io/sssom/dev/

## IRI structure

| IRI | Contents |
| --- | -------- |
| `https://w3id.org/sbom/` | Bridge concept scheme + ontology declaration |
| `https://w3id.org/sbom/mapping/` | SSSOM mapping set files |
| `https://w3id.org/sbom/req/ntia/` | NTIA SBOM Minimum Elements concept scheme |
