# gcnx (w3id.org/gcnx)

Geoconnex provides stable identifiers under `https://w3id.org/gcnx/`,
for Geoconnex Peramanent Idenifiers. The Geoconnex project provides
technical infrastructure and guidance for creating an open,
community-contribution model for a knowledge graph linking hydrologic
features in the United States, published in accordance with Spatial Data
on the Web best practices as an implementation of Internet of Water principles.

## Redirect behavior

The Geoconnex namespace namespace is managed at 
https://github.com/internetofwater/geoconnex.us.
The w3id will follow two redirects as demonstrated in the first example.

Examples:

- `https://w3id.org/gcnx` -> `https://geoconnex.us` -> `https://docs.geoconnex.us`
- `https://w3id.org/gcnx/ref/states/08` -> `https://geoconnex.us/ref/states/08`
- `https://w3id.org/gcnx/ref/mainstems/29559` -> `https://geoconnex.us/ref/mainstems/29559`
