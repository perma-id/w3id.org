Abstract

The purpose of this vocabulary is to describe in a semantic manner information related to university centers, which are used to teach classes or has research staff. Regarding the center, it contains information about the school, the faculties, the study centers, the research institutions, the hospitals, etc.

Introduction back to ToC

This vocabulary represents information related to university centers. The data used in it comes from a set of public spanish universities, more specifically, they belong to: the Automonous University of Madrid (UAM), the Carlos III University of Madrid (UC3M), the Complutense University of Madrid (UCM), the King Juan Carlos University (URJC), the University of Valladolid (UVa) and the University of Huelva (UHU). In this way, this vocabulary describes information about a center and its university, municipality and campus associated.

Moreover, it uses the geo and geosparql vocabularies to represent the geographical location of the center and its university associated, according to different coordinate reference systems commonly used for this type of data. Additionally, it also uses the territory vocabulary to describe territorial entities in Spain, for the representation of the municipality where the center is located in. Finally, it also reuses the organizations vocabulary to represent the center entity and its university associated.

Vocabulary for the representation of university centers: Description back to ToC

Figure 1 shows the classes and properties of the vocabulary that represents the data of university centers:

As mentioned before, the scope of this vocabulary is the representation of data from university centers. To accomplish this purpose, as it can be seen in the figure, several classes and properties have been defined in order to describe these data, such as the Centro class, which has several SKOS concept schemes to represent the type of center, its nature and its current situation. Moreover, it has been reused the class and properties of esadm:Municipio vocabulary, for the representation of the municipality in which the center is located in as well as the belonging campus. Additionally, it has been also reused the vocabulary of org:Organization to show how the class Universidad has a suborganization, which is the Centro class. In this way, this class has been reused to explain that both mentioned classes (Universidad and Centro) have a geographical location, represented by the vocabulary geosparql.

Examples of use

In order to understand the use of some classes and properties in this vocabulary some examples have been provided:

      univCentro:ETSIINF a univCentro:Centro ;
         dct:identifier "28027114" ;
         univCentro:fechaDesdeSituacionCentro "19760701"^^xsd:date ;
         univCentro:tipoCentro <http://vocab.universidadesabiertas.es/kos/tipo-centro/centroPropio> ;
         univCentro:naturalezaCentro <http://vocab.universidadesabiertas.es/kos/naturaleza-centro/publico> ;
         univCentro:situacionCentro <http://vocab.universidadesabiertas.es/kos/situacion-centro/activo> ;
         esadm:municipio esadm:Boadilla_del_Monte ;
         univCentro:estaUbicado univCentro:Campus_de_Montegancedo ;
         org:hasSite univCentro:ETSIINF_location ;
         org:subOrganizationOf univCentro:UPM ;
         rdfs:subClassOf org:Organization .
         
      univCentro:UPM a univCentro:Universidad ;
         dct:identifier "025" ;
         univCentro:tieneAsociado univCentro:2021_22 ;
         org:hasSubOrganization univCentro:ETSIINF ;
         rdfs:subClassOf org:Organization .
   
      univCentro:2021_22 a univCentro:CursoAcademico .
   
      esadm:Boadilla_del_Monte a esadm:Municipio ;
         dct:identifier "022" .
   
      univCentro:Campus_de_Montegancedo a univCentro:Campus;
         dct:identifier "02214" ;
         esadm:municipio esadm:Boadilla_del_Monte .
   
      univCentro:ETSIINF_location geosparql:hasGeometry univCentro:ETSIINF_geometry .
   
      univCentro:ETSIINF_geometry a geosparql:Point ;
         geo:long "-3.83561"^^xsd:double ;
         geo:lat "40.40575"^^xsd:double ;
         geosparql:asWKT "POINT (-3.83561,40.40575)"^^geosparql:wktLiteral .
