# AnIML Ontology

The AnIML (Analytical Information Markup Language) Ontology.

**Maintainer**: 
Jacopo de Berardinis (jacodb@liverpool.ac.uk | @jonnybluesman)

**Repository**:
https://github.com/KE-UniLiv/animl-ontology

## URI Patterns

* `https://w3id.org/animl/ontology/` - Resolves to the latest version of the ontology.
* `https://w3id.org/animl/ontology/1.0.0` - Resolves to version 1.0.0.
