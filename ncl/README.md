# NCL: an ontology for describing food products and consumer representations of naturalness 
===========================================================================================

# Description and publication 

The NATCL’INN project aims to help agri-food companies better meet consumers' expectations for naturalness, which is perceived as a guarantee of quality, health and taste. Based on the characterisation of representations of naturalness and the identification of relevant markers, it is developing a multi-criteria arbitration tool that integrates technical, economic and regulatory constraints. Led by ADRIA, UBO-LEGO, INRAE and several industrial partners, this collaborative project has been certified by Valorial.
    .  


## Persistent URI 
https://w3id.org/ncl is a persistent URI namespace for **Natcl'inn Ontology (NCL)** Ontology resources.
