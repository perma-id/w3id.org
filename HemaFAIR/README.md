# HemaFAIR

This repository contains the coniguration for the [W3ID](https://w3id.org/) persistent identifier for the HemaFAIR project. 

## Project Overview

HemaFAIR is an international collaborative initiative dedicated to making hematological rare disease registry (meta)data FAIR (Findable, Accessible, Interoperable, and Reusable). By leveraging FAIR data and standards in research and education, the project enhances the scientific profile of the Cyprus Institute of Neurology & Genetics (CING) and strengthens the position of researchers in Cyprus within the fields of biomedical informatics and rare hematological diseases.

## Redirects

The following namespaces are registered under this directory:
- **Project Website**: [https://w3id.org/hemafair](https://w3id.org/hemafair)  
  Redirects to the official project [homepage](https://hemafairproject.eu/).
- **FAIR Data Point (FDP)**: [https://w3id.org/hemafair/fdp](https://w3id.org/hemafair/fdp)  
  Redirects to the HemaFAIR [FDP instance](https://fdp.ithanet.eu/). All sub-paths (e.g., `/fdp/catalog/any-id`) are maintained and redirected to the corresponding metadata resources.
