# ALLMaC / OPO Consolidation

This directory contains the configuration for the permanent identifier:

- [**https://w3id.org/allmac/opo-consolidation**](https://w3id.org/allmac/opo-consolidation)

It currently redirects to the GitHub repository:

- [**https://github.com/SandraSchaftner/orkg-properties-ontology-consolidation**](https://github.com/SandraSchaftner/orkg-properties-ontology-consolidation)

The **OPO Consolidation** module is part of the **ALLMaC – Agentic LLM-augmented Construction of Scientific Knowledge Graphs (SKGs)** framework.
It implements a semi-automated workflow that consolidates the existing ORKG property set into the **ORKG Properties Ontology (OPO) Consolidated** by combining lexical deduplication, embedding-based semantic clustering, and LLM-assisted canonical selection and quality filtering.

The resulting ontology is evaluated on a Gold Standard of 153 research papers and aims to improve schema conciseness and semantic consistency while preserving backward compatibility through explicit mappings from deprecated properties to their canonical counterparts.

## Intended usage

The identifier [**https://w3id.org/allmac/opo-consolidation**](https://w3id.org/allmac/opo-consolidation) provides a stable, citable URL for:

- the OPO consolidation scripts and configuration, 
- the consolidated ORKG Properties Ontology (OPO) releases and mapping files, 
- future updates and evaluations of the OPO consolidation workflow as part of the ALLMaC pipeline.
