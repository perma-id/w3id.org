# namo

## Purpose

Permanent W3 identifiers for the NAMO (New Approach Methodology Ontology and Schema) project — a LinkML-based schema for modeling new approach methodologies (NAMs) in biomedical research, including animal models, cell lines, organoids, organ-on-chip systems, and related model systems.
