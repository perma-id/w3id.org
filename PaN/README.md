# PaN Ontologies

Persistent identifiers for Photon and Neutron (PaN) ontologies

## Description

This repository manages persistent HTTP identifiers (PIDs) under `w3id.org/PaN/` for ontologies developed within the Photon and Neutron (PaN) community, providing stable, resolvable, URIs for:
- experimental techniques.
