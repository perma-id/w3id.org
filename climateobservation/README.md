# climateobservations

Persistent identifiers for the E-OBS Knowledge Graph and EOBS-Agent.

## Identifiers

### https://w3id.org/climateobservations/eobs-v31

SPARQL endpoint for the E-OBS Knowledge Graph:

https://obs.webapps.nfdi4earth.de/sparql/

### https://w3id.org/climateobservations/eobs-agent

Web-based natural language interface for querying the E-OBS Knowledge Graph.
