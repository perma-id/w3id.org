IMX Specifications
=========================

* https://imx-extension.lab.gaia-x.eu/development/docs#/
* [IMX Documents](https://imx-extension.lab.gaia-x.eu/development/docs#/)

Endpoints:
  * with `Accept: text/turtle` returns the IMX Shacl shapes
  * with `Accept: application/rdf+xml` returns the IMX OWL ontology
