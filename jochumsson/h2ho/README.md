# Hydrogen Hazard Ontology (H2HO)

Maintainer: Thorvaldur Jochumsson  
GitHub ID: jochumsson

This namespace provides persistent identifiers for the Hydrogen Hazard Ontology (H2HO),
an abstract hydrogen safety ontology that extends the HO61508 hazard ontology.
