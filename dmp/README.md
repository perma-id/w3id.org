## OSTrails DMP Evaluation Commons

Persistent identifiers for the DMP Common Standard Compliance (DMP-CSC)
evaluation metrics and tests, part of the [OSTrails project](https://ostrails.eu)
(Open Science Plan-Track-Assess Pathways).

### Metrics

`https://w3id.org/dmp/evaluation/metric/<metric-id>` redirects to the
corresponding metric section on the
[DMP Evaluation Metrics](https://docs.ostrails.eu/en/latest/commons/dmp/dmp-evaluation-metrics.html)
page.

Example: `https://w3id.org/dmp/evaluation/metric/data.reused.co.1` redirects to
[Metric 1: Reused Dataset Declared in the DMP](https://docs.ostrails.eu/en/latest/commons/dmp/dmp-evaluation-metrics.html#dmp-metric-1).

### Tests

`https://w3id.org/dmp/evaluation/test/T-DCSC-NNN` redirects to the
corresponding test section on the
[DMP Catalogue of Tests](https://docs.ostrails.eu/en/latest/commons/dmp/dmp-catalogue-of-tests.html)
page.

Example: `https://w3id.org/dmp/evaluation/test/T-DCSC-001` redirects to
[Test 1: Check for reused dataset declaration](https://docs.ostrails.eu/en/latest/commons/dmp/dmp-catalogue-of-tests.html#dmp-test-1).
