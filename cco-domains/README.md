# cco-domains

Permanent identifier space for the CCO Domain Ontologies.

## Purpose
This namespace provides stable, long-term IRIs for ontology modules located in the
`ontology/domains/` and `ontology/cco/` directories of the canonical repository:

https://github.com/cco-domains/cco-domains

## Redirect Structure
- `https://w3id.org/cco-domains/domains/...`  
  → `ontology/domains/...`

- `https://w3id.org/cco-domains/cco/...`  
  → `ontology/cco/...`

These redirects point to the raw content hosted at:

https://raw.githubusercontent.com/cco-domains/cco-domains/refs/heads/main/ontology/
