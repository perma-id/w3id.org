# Know Your Agent (KYA) Standard

This directory contains the redirection logic for the **Know Your Agent (KYA)** autonomous agent manifest standard.

- **Maintained by:**
    * Open KYA (https://github.com/open-kya)
    * [Lars Kæraa Lücke](https://github.com/WP-LKL), [HUV.AI](https://HUV.AI)
- **Primary Repository:** [open-kya/kya-standard](https://github.com/open-kya/kya-standard)
- **Contact:** standards@huv.ai (or via GitHub issues in the primary repository)

## Redirects
- `https://w3id.org/kya/v1` -> JSON-LD Context & Vocabulary
- `https://w3id.org/kya/v1/schema` -> JSON Schema
- `https://w3id.org/kya/spec` -> Formal Specification
