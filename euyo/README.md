# /ssv/
This [W3ID](https://w3id.org) provides a persistent URI namespace for the European Union Youth Orchestra Concert Explorer, developed as part of the research project "Composing Change: Orchestral Impact in European Societies" funded by Land Niederösterreich, Abteilung Wissenschaft und Forschung.

## Uses

This namespace will be used to point to the EUYO Concert Explorer interactive data visualisation application for musicology.
