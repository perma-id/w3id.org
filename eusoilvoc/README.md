# eusoilvoc

Persistent w3id namespace for the EUSO soil vocabulary.

Persistent base IRI: [https://w3id.org/eusoilvoc/](https://w3id.org/eusoilvoc/)

RDF namespace IRI: [https://w3id.org/eusoilvoc#](https://w3id.org/eusoilvoc#)
