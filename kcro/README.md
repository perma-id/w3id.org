# KCRO — Kubernetes Cybersecurity Research Ontology

**w3id.org/kcro**

A gUFO-grounded OWL 2 ontology of Kubernetes security misconfigurations,
modelling resource kinds, security aspects (dispositions and qualities),
inter-resource relators, and a COVER-style risk layer. Developed for a BSc
thesis at the University of Twente, grounded in the KubeObjects corpus
(75,390 manifests).

- Namespace: `https://w3id.org/kcro#` (hash IRIs)
- Content negotiation: RDF accept headers → `kcro.ttl` (raw); default → the GitHub repository
