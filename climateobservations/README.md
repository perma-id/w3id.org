# climateobservations

Persistent identifiers for the E-OBS Knowledge Graph and EOBS-Agent.

## Identifiers

### https://w3id.org/climateobservations/eobs-v31

Redirects to the E-OBS v31 Knowledge Graph SPARQL endpoint.

### https://w3id.org/climateobservations/eobs-agent

Redirects to the EOBS-Agent natural-language querying interface.
