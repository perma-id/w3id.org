# RCAO Ontology

[https://w3id.org/rcao/](https://w3id.org/rcao/) -> [https://vicomtech.github.io/rcao/versions/latest/index-en.html](https://vicomtech.github.io/rcao/versions/latest/index-en.html)


**Resilient Collective Awareness Ontology (RCAO)** provides a formal model for representing collective awareness in the autonomous agents and driving research domain. This work is fully supported by the EU AIGREGGATE ([https://cordis.europa.eu/project/id/101202457](https://cordis.europa.eu/project/id/101202457)] project funded from the European Union´s Horizon 2027 research and innovation program under the grant agreement no ```10.3030/101202457```.
