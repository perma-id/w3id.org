# w3id.org/worldspec

Permanent identifiers for the **WorldSpec** specification — a declarative
language for modeling software systems as entities, relationships, states,
invariants, actions, and transitions.

`https://w3id.org/worldspec/*` redirects (HTTP 302) to the published
specification site so that the identifiers remain stable even if the hosting
location changes.

## Identifiers

| w3id.org URL | Resolves to |
|--------------|-------------|
| `https://w3id.org/worldspec/` | Project / spec home |
| `https://w3id.org/worldspec/spec/` | Latest specification version |
| `https://w3id.org/worldspec/spec/0.1/` | Specification v0.1 |
| `https://w3id.org/worldspec/ns/0.1#` | Namespace document v0.1 |
| `https://w3id.org/worldspec/schemas/worldspec-model-0.1.schema.json` | Model JSON Schema v0.1 |

Current redirect target: <https://banuap.github.io/worldspec/>
