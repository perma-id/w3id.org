# /chint/
This [W3ID](https://w3id.org) provides a persistent URI namespace for the Cultural Heritage Interactions (CHint) ontological framework and its documentation

## Uses
This namespace is broken down into different different parts:
* [https://w3id.org/chint/docs/](https://w3id.org/chint/docs/) leads to the documentation the ontology.
* [https://w3id.org/chint/ontology/](https://w3id.org/chint/ontology/) leads to a raw owl file of the ontology (useful for imports)
* [https://w3id.org/chint/development/](https://w3id.org/chint/development/) leads to a github repository that contains the development documentation of the ontology.
