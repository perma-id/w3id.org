# Harbour Credentials Ontology

Harbour is a library and ontology for issuing and verifying W3C Verifiable Credentials with selective disclosure (SD-JWT-VC), designed for the Gaia-X ecosystem.

## Homepage

- <https://reachhaven.github.io/harbour-credentials/>

## Documentation

Refer to the documentation site for the schema reference and API docs:

- <https://reachhaven.github.io/harbour-credentials/reference/schemas/>

## URL Patterns

- `https://w3id.org/reachhaven/harbour/{domain}/v{version}/` — Ontology namespace (for prefix)
- `https://w3id.org/reachhaven/harbour/{domain}/v{version}` — Ontology IRI (for owl:Ontology)
- `https://w3id.org/reachhaven/harbour/{domain}/v{version}/shapes` — SHACL shapes
- `https://w3id.org/reachhaven/harbour/{domain}/v{version}/context` — JSON-LD context

Domains: `core`, `gx`, `delegate`

## Resolution

W3ID IRIs resolve to static files published on GitHub Pages via the release workflow.
