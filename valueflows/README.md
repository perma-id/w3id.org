# /valueflows/

This [W3ID](https://w3id.org) provides a persistent URI namespace
for [Valueflows (VF)][VF] resources.

## Uses

This namespace is currently only used to host RDF ontologies.
The mian one being:

<https://w3id.org/valueflows/ont/vf>
