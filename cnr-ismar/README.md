
# CNR-ISMAR Persistent Identifiers

This namespace provides persistent identifiers managed by the Institute of Marine Sciences (ISMAR) of the National Research Council of Italy (CNR).

The purpose of this namespace is to provide stable, long-term, and resolvable identifiers for semantic resources developed and maintained by CNR-ISMAR. The identifiers are intended to remain stable independently of the underlying hosting platform or repository structure.

## Namespace

Base URI:

https://w3id.org/cnr-ismar/

## Resource Categories

The namespace currently supports the following categories of resources.

### Metadata Profiles

Metadata profiles, application profiles, guidelines, and related semantic artefacts.

URI pattern:

https://w3id.org/cnr-ismar/profile/{resource}

### Ontologies

Domain ontologies and ontology extensions developed or maintained by CNR-ISMAR.

URI pattern:

https://w3id.org/cnr-ismar/ontology/{resource}

### Sensor Templates

Reusable sensor descriptions and sensor model templates intended to support interoperability, cataloguing, and semantic annotation of observing systems.

URI pattern:

https://w3id.org/cnr-ismar/template/{resource}

### Sensor Deployments

Descriptions of deployed sensors, observing platforms, and related operational resources represented using SOSA/SSN and associated ontologies.

URI pattern:

https://w3id.org/cnr-ismar/deployment/{resource}

## Persistence Policy

CNR-ISMAR is committed to maintaining the stability of the identifiers published within this namespace.

The physical location of the referenced resources may change over time as repositories, infrastructures, and services evolve. Such changes should not affect the published identifiers. Redirect targets may therefore be updated when necessary to ensure long-term accessibility while preserving identifier persistence.
