# biodivrisK-onto

Persistent URI namespace for BiodivRisk-Onto, an OWL 2 ontology for semantic alignment of biodiversity risk disclosure frameworks (TNFD, CSRD/ESRS E4, SBTN and GRI 101) for Latin America and the Caribbean.
