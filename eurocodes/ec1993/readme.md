# E3O - Eurocode 3 Ontology

<p> This repository contains the files of the E3O - Eurocode 3 Ontology. </p>
<p> The Eurocode 3 Ontology (E3O) provides a semantically structured representation of the European structural steel design standards, known as Eurocode 3. E3O extends the Eurocode Core Ontology (http://www.w3id.org/eurocodes/ec1990#) to include specifics from Eurocde 3, thus contributing to generate a complete machine-readable version of the Eurocodes.</p>

## Ontology
* Doc: https://cramonell.github.io/eurocodes/ec1993/actual/index-en.html
* Turtle: https://cramonell.github.io/eurocodes/ec1993/actual/ontology.ttl
