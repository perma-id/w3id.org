# ECO - Eurocode Core Ontology

<p> This repository contains the files of the E3O - Eurocode 3 Ontology. </p>
<p> The Eurocode 3 Ontology (E3O) provides a semantically structured representation of the European structural steel design standards known as EN1993 or Eurocode 3. E3O extends ECO (http://www.w3id.org/eurocodes/ec1990#) to include the specifications of the Eurocde 3 into a machine readable version of the Eurocodes.</p>
