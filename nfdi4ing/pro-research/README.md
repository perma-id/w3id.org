Redirections:
* HTML: Visit documentation
  * `https://w3id.org/nfdi4ing/pro-research`
  * --> `https://nfdi4ing.pages.rwth-aachen.de/pro-research/pro-research-ontology`
* Turtle, RDF/XML, N-Triples, JSON-LD: Access respective ontology file
  * via content negotiation
    * `https://w3id.org/nfdi4ing/pro-research` with HTTP-Header `Accept: text/turtle` for Turtle (`.ttl`), `Accept: application/rdf+xml` for RDF/XML (`.xml`), `Accept: text/n3` for N-Triples (`.nt`), `Accept: application/ld+json` for JSON-LD (`.jsonld`), or ...
    * --> `https://nfdi4ing.pages.rwth-aachen.de/pro-research/pro-research-ontology/ontology.ttl` (or `.xml`/`.nt`/`.jsonld`) 
  * or via direct redirect of the filename
    * `https://w3id.org/nfdi4ing/pro-research/ontology.ttl` (or `.xml`/`.nt`/`.jsonld`)
    * --> `https://nfdi4ing.pages.rwth-aachen.de/pro-research/pro-research-ontology/ontology.ttl` (or `.xml`/`.nt`/`.jsonld`)
