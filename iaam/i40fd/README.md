# Industry 4.0 Field Device Ontology

This ontology provides concepts that allow interoperability at the application layer of field devices (e.g., IO-Link, PROFINET, CANOpen)
and allows a generalization based on device profiles, units of measurement, and communication properties.

This is a research project of the [Institute for Applied Automation and Mechatronics](https://www.iaam.fh-aachen.de) (IaaM).
