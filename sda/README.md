# /sda/
This [W3ID](https://w3id.org) provides a persistent URI namespace for the **Scholarly Domain Activities Ontology**. 

## Uses

This namespace is used solely for the **Scholarly Domain Activities Ontology**.

The path on its own, or with the suffix `/latest`, will always point to the latest version of the ontology (e.g., `https://w3id.org/sda/1.0/`).

Content negotiation is used to choose a suitable RDF serialisation format.
