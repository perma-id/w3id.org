# /ssv/
This [W3ID](https://w3id.org) provides a persistent URI namespace for data produced by [Scholarly Domain Activities Ontology](https:   ) and associated projects.

## Uses

This namespace is used solely for data related to the **Scholarly Domain Activities Ontology**.

The path on its own, or with the suffix `/latest`, will always point to the latest version of the ontology. 

Content negotiation is used to choose a suitable RDF serialisation format.
