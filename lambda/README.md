# LAMBDA-BER-SCHEMA

## Purpose

Persistent identifiers for the **LAMBDA-BER-SCHEMA** project - a LinkML schema for representing structural biological imaging data and experiments from the Berkeley Lab (BER) research programs.

The schema provides a unified data model for:
- Cryo-EM (cryo-electron microscopy)
- X-ray crystallography
- SAXS/WAXS (small/wide-angle X-ray scattering)
- SANS (small-angle neutron scattering)
