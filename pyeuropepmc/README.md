# pyeuropepmc

This is the permanent identifier for PyEuropePMC RDF data.

## Description

PyEuropePMC is a Python toolkit for searching, retrieving, and analyzing scientific literature from Europe PMC. This w3id.org identifier provides resolvable URIs for RDF data generated by the library, including entities like papers, authors, and institutions.

## Redirection

All requests to `https://w3id.org/pyeuropepmc/*` are redirected to the corresponding files in the GitHub repository's raw data: `https://raw.githubusercontent.com/JonasHeinickeBio/pyeuropepmc/main/data/*`.

This allows RDF URIs generated by PyEuropePMC to be resolvable linked data.
