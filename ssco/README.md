# /ssco/

This namespace provides persistent identifiers for the SSCO ontology.

## Namespace

- `https://w3id.org/ssco#` — Root namespace of the SSCO ontology.
- `https://w3id.org/ssco/core#` — Core SSCO module.
- `https://w3id.org/ssco/bfi#` — BIM–FEM integration module.
- `https://w3id.org/ssco/scc#` — Sustainability and compliance-checking module.

## Redirect targets

- HTML documentation: `https://shihang-z.github.io/SSCO/`
- Root ontology, Turtle: `https://shihang-z.github.io/SSCO/ontology.ttl`
- Root ontology, RDF/XML or OWL: `https://shihang-z.github.io/SSCO/ontology.owl`
- Root ontology, JSON-LD: `https://shihang-z.github.io/SSCO/ontology.jsonld`
- Root ontology, N-Triples: `https://shihang-z.github.io/SSCO/ontology.nt`
- Core module: `https://shihang-z.github.io/SSCO/ssco-core.ttl`
- BIM–FEM integration module: `https://shihang-z.github.io/SSCO/ssco-bfi.ttl`
- Sustainability and compliance-checking module: `https://shihang-z.github.io/SSCO/ssco-scc.ttl`
