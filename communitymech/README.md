# communitymech

## Purpose

Permanent W3 identifiers for the CommunityMech project — a LinkML-based knowledge base for modeling microbial community structure, function, and ecological interactions.

## Links

- **Homepage:** https://github.com/CultureBotAI/CommunityMech
- **Schema Documentation:** https://CultureBotAI.github.io/CommunityMech/
- **LinkML Schema:** https://raw.githubusercontent.com/CultureBotAI/CommunityMech/main/src/communitymech/schema/communitymech.yaml

## Example URIs

- `https://w3id.org/communitymech/` — base namespace
- `https://w3id.org/communitymech/MicrobialCommunity` — schema class
- `https://w3id.org/communitymech/communitymech.yaml` — LinkML schema artifact
