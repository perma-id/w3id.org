# w3id.org/sanctions

Persistent URI namespace for the Sanctions Legislative Implementation Vocabulary (SLIV).

## Maintainer

- **Name:** Mariam Hadjilouka
- **GitHub:** https://github.com/Mariam-the-Solivagus
- **Email:** mariamhadjilouka@icloud.com

## Description

SLIV is an OWL vocabulary extending the UN System Document Ontology (UNDO) 
for linking UN Security Council sanctions resolutions to their EU implementing 
regulations and designated entities.
