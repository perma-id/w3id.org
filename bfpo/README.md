# BFPO — Bioreactor Fermentation Process Ontology

Namespace: https://w3id.org/bfpo#

Ontología de dominio para la formalización del conocimiento del 
bioproceso de fermentación en biorreactor. Desarrollada como parte 
del proyecto CDT+KG en la Universidad del Cauca, Colombia.

**BioPortal:** https://bioportal.bioontology.org/ontologies/BFPO  
**Institución:** Universidad del Cauca.
