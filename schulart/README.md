# /schulart/
This [W3ID](https://w3id.org) provides a persistent URI namespace for vocabularies related to the German school types. 

see [https://fwu-de.github.io/schulart-ontologie](https://fwu-de.github.io/schulart-ontologie) and [https://github.com/FWU-DE/schulart-ontologie](https://github.com/FWU-DE/schulart-ontologie)
